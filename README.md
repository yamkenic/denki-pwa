# 電気設備概算算出ツール — 公開手順

このフォルダ一式をそのままアップロードするだけで、スマホ・PCどちらからでも使えるWebアプリ（PWA）として公開できます。

---

## ファイル構成

```
denki-pwa/
├── index.html        ← メインのツール画面
├── manifest.json     ← PWA設定（アプリ名・アイコン等）
├── sw.js             ← Service Worker（オフライン対応）
└── icons/
    ├── icon-192.png  ← アプリアイコン（小）
    └── icon-512.png  ← アプリアイコン（大）
```

---

## 方法① Netlify（最も簡単・推奨）

1. https://app.netlify.com にアクセス
2. Googleアカウント等でサインアップ（無料）
3. ダッシュボードの **「Add new site」→「Deploy manually」** をクリック
4. この `denki-pwa` フォルダをまるごとドラッグ＆ドロップ
5. 数秒で `https://xxxxxxxx.netlify.app` というURLが発行される

→ そのURLをブックマークまたはQRコードにして共有するだけで完了です。

---

## 方法② GitHub Pages

1. https://github.com でアカウント作成（無料）
2. 新しいリポジトリを作成（例：`denki-tool`）
3. このフォルダ内のファイルをすべてアップロード
4. リポジトリの **Settings → Pages → Branch: main** を選択して Save
5. `https://あなたのID.github.io/denki-tool/` で公開される

---

## スマホへのインストール方法（利用者向け）

### iPhoneの場合
1. Safariでツールのページを開く
2. 下部の「共有」ボタン（四角に矢印）をタップ
3. 「ホーム画面に追加」→「追加」

### Androidの場合
1. Chromeでツールのページを開く
2. 右上メニュー（…）→「ホーム画面に追加」
3. または、アドレスバー下に自動で「アプリをインストール」バナーが表示される

---

## オフライン動作について

一度アクセスすると、次回以降はインターネット接続なしでも使用できます。
（計算はすべてブラウザ内で完結しているため、サーバー通信不要）
